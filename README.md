## Detailed Lab Information

**Objective:**  
- Configure and manage virtual machine networks using Hyper-V on Windows Server 2019.  
- Test connectivity between virtual machines and the Hyper-V host.  
- Implement network isolation and control inbound/outbound traffic for security.  

**Scenario / Background:**  
Virtualization allows multiple operating systems to run on the same physical hardware. This lab simulates real-world networking environments. Tasks include:  
- Creating internal and external virtual networks  
- Connecting virtual machines and verifying network connectivity  
- Applying network isolation rules to secure resources  
- Reviewing network adapter configurations and rules  

**Skills Gained:**  
- Virtualization network setup and management  
- Network isolation for security  
- Troubleshooting connectivity issues  
- Understanding inbound/outbound traffic rules
